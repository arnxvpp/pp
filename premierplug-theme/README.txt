== Description ==

PremierPlug is a custom WordPress theme designed for modern media agencies. Features include:

* Animated homepage intro sequence
* Overlay navigation menu with multi-level support
* Responsive design for all devices
* Custom page templates
* WordPress Customizer integration
* Translation ready

== Installation ==

1. Upload the theme folder to /wp-content/themes/ directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Go to Appearance > Customize to configure theme settings
4. Create your menus at Appearance > Menus:
   - Create a "Primary Menu" and assign it to "Primary Menu (Overlay)" location
   - Create a "Footer Menu" and assign it to "Footer Menu" location
