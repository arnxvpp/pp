## Quick Start

1. **Upload theme:**
   ```bash
   Upload premierplug-theme/ to /wp-content/themes/
   ```

2. **Activate:**
   - WordPress Admin → Appearance → Themes → Activate PremierPlug

3. **Configure:**
   - Create menus (Appearance → Menus)
   - Create pages for services
   - Set featured images

4. **Test:**
   - Visit homepage
   - Check navigation works
   - Test on mobile
